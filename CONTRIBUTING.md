# Contributing to TooltrayTyper

Thanks for your interest in contributing.

## How to contribute

1. Fork the repo
2. Create a feature branch
3. Make focused changes
4. Open a pull request with a clear description

## Development

```bash
npm install
npm start
```

## Guidelines

- Keep changes small and reviewable
- Document behavior changes in README when needed
- Avoid committing secrets or API keys
- Test tray startup and hotkey flow after UI/main-process changes

## Reporting issues

Please include:

- OS version
- Node/Electron versions
- Steps to reproduce
- Expected vs actual behavior
- Logs/screenshots where useful
